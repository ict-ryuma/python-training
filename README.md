## 📘 2025-05-17(土) 学習記録

### ✅ 取り組んだこと

- `DictReader` を使って **CSVファイルから辞書形式でデータを読み取り**
- `if / elif / else` による **年齢による分類（若手／中堅／ベテラン）**
- `"エンジニア" in 職業` などの **文字列検索で職業カテゴリ（開発／デザイン／事業）**を分類
- `defaultdict(int)` を使った **集計処理（初期値0が自動で便利）**
- **複数条件の処理（職業 × 年齢）をクロス集計**し、出力できるようになった
- `f"{変数}"` での **フォーマット文字列**、`\n` での改行の意味と使い方も理解
- `for key, value in dict.items()` を使った **辞書のループ処理**の基本と意図を習得

### 🎯 得た気づき・理解

- `defaultdict(int)` を使えば、初期値 `0` をいちいち定義しなくてもOK！
- `int()` を使わないと `"年齢"` が文字列のままで条件判定できない
- `row["キー名"]` でCSVの列データにアクセスできる（名前は定義不要）
- `"文字列" in 値` で、部分一致判定ができる（"エンジニア" in 職業）
- `for key, count in dict.items()` は辞書を使ったループで最も基本的な書き方
